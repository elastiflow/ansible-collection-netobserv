# Ansible Role: elastiflow.netobserv.netobserv_flow
Version: 0.0.1

NetObserv Flow role for ElastiFlow - installs and configures the flow collector.


## Requirements

| Platform | Versions |
| -------- | -------- |
| EL | 8, 9 |
| Ubuntu | 22.04, 24.04 |
